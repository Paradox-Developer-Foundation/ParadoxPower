# ParadoxPower

A Paradox parser, used for parsing Paradox script files
一个解析器, 用来解析 Paradox 脚本文件

## Supported Games

- Europa Universalis IV (欧陆风云4)
- Hearts of Iron IV (钢铁雄心4)
- Victoria (维多利亚)
- Stellaris (群星)
